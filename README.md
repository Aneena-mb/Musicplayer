# 🎵 Music Player

A modern web-based music player built using HTML, CSS, and JavaScript. The application allows users to play music from a predefined playlist with an interactive and visually appealing user interface.

## Features

* ▶️ Play and Pause songs
* ⏭️ Next and Previous track controls
* 🔊 Volume control
* 💿 Rotating album artwork while music is playing
* ⏱️ Progress bar with song duration tracking
* 🎨 Dark theme with gradient background
* 🔄 Automatic playback of the next song
* 📱 Responsive and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Future Enhancements

* Playlist sidebar
* Shuffle and repeat functionality
* Keyboard shortcuts
* Dynamic background based on album art
* User-uploaded music support
* Music metadata display (artist, album, genre)
